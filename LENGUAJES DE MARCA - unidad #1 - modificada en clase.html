CSS: cascading style sheets

cómo aplicamos el aspecto visual

Aplicamos estilo al contenido HTML 

/* margen al cuerpo del documento */
body {
  padding: 40px;
}

/* color cabecera 1 */
h1 {
  color: black;
  background-color: red;
}

/* color cabecera 2 */
h2 {
  color: green;
  background-color: silver;
}
  
 /* element: html tags */
 element {
    attribute: value;
 }
 /* class: clase asignada a un tag html mediante class="name" */
 .class {
    attribute: value;
 }
 /* id: identificador asignado a un tag html mediante id="identifier" */
 #id {
    attribute: value;
 }
  • colores: https://www.w3schools.com/colors/colors_names.asp
  • reglas de precedencia: inline, IDs, classes attributes pseudo-classes, elements pseudo-elements
